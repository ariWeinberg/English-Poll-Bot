## About

**RU:** GREEN-API — международная IT-компания с офисами в России, Казахстане, Бразилии и Израиле, предоставляющая доступные API для мессенджеров: WhatsApp, WhatsApp Business (WABA), Telegram и MAX. Более 9 лет обеспечиваем стабильную работу по всей планете. Наши пользователи отправляют и получают более 2 миллионов сообщений в день через наш сервис. Работаем с партнерами в более чем 48 странах мира. Компания предлагает единый интерфейс API для работы с несколькими мессенджерами, что позволяет разработчикам и бизнесу подключить все каналы коммуникации через одну платформу.

**EN:** GREEN-API is an international IT company with offices in Russia, Kazakhstan, Brazil, and Israel, providing affordable APIs for messengers: WhatsApp, WhatsApp Business (WABA), Telegram, and MAX. For over 9 years, we have ensured stable operations worldwide. Our users send and receive more than 2 million messages daily through our service. We work with partners in over 48 countries. The company offers a unified API interface for multiple messengers, allowing developers and businesses to connect all communication channels through a single platform.

## Frequently Asked Questions

### Top Questions

**Q: Какие мессенджеры поддерживает GREEN-API?**
**A:** GREEN-API поддерживает WhatsApp (основной API и официальный WABA), Telegram и MAX через единую платформу.
**Details:** [green-api.com](https://green-api.com/)

**Q: Как отправить файл через API?**
**A:** Используйте методы SendFileByUrl для внешних файлов или SendFileByUpload для локальных файлов. Методы доступны для WhatsApp, Telegram и MAX.
**Details:** [green-api.com/docs/faq/how-to-send-file/](https://green-api.com/docs/faq/how-to-send-file/) | [EN](https://green-api.com/en/docs/faq/how-to-send-file/)

**Q: Какие типы файлов поддерживает API?**
**A:** Поддерживаются jpeg, png, mp4, mp3, docx, pdf и другие популярные форматы до 100МБ.
**Details:** [green-api.com/docs/faq/types-of-extensions-supported-by-API/](https://green-api.com/docs/faq/types-of-extensions-supported-by-API/) | [EN](https://green-api.com/en/docs/faq/types-of-extensions-supported-by-API/)

**Q: Как защитить номер от блокировки?**
**A:** Прогревайте номер, персонализируйте сообщения, используйте интервалы между отправками.
**Details:** [green-api.com/docs/faq/how-to-protect-number-from-ban/](https://green-api.com/docs/faq/how-to-protect-number-from-ban/) | [EN](https://green-api.com/en/docs/faq/how-to-protect-number-from-ban/)

**Q: В чём разница между GREEN-API и WABA?**
**A:** GREEN-API — неофициальный API для WhatsApp (дешевле, быстрее подключение, без модерации контента). WABA — официальный WhatsApp Business API с шаблонами, кнопками и верифицированным аккаунтом.
**Details:** [green-api.com/waba](https://green-api.com/waba)

**Q: How to use proxy server in WhatsApp?**
**A:** Configure proxy settings to access WhatsApp in restricted regions using GREEN-API free proxy.
**Details:** [green-api.com/en/docs/faq/how-to-use-a-proxy-server-in-whatsapp/](https://green-api.com/en/docs/faq/how-to-use-a-proxy-server-in-whatsapp/)

**Q: How to send emoji via API?**
**A:** Include Unicode emoji characters directly in your message text when using SendMessage method.
**Details:** [green-api.com/en/docs/faq/how-to-send-emoji/](https://green-api.com/en/docs/faq/how-to-send-emoji/)
